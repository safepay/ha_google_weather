# Google Weather Integration

Integrate Google Weather API with Home Assistant using your existing Google OAuth credentials.

## Key Features

✅ Reuses existing Google OAuth credentials  
✅ Configurable entity ID prefixes  
✅ Weather forecasts and current conditions  
✅ Weather warning binary sensors  
✅ Observational sensors (UV, visibility, precipitation)

## Quick Start

1. Enable Google Weather API in Google Cloud Console
2. Add this integration in Home Assistant
3. Authenticate with your Google account
4. Configure location and entity prefix
5. Done!

See the full README for detailed setup instructions.
